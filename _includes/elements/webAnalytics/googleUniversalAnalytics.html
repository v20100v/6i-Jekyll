{% comment %} Google Analytics - Universal Analytics : analytics.js ===================================================== The analytics.js library (also known as "the Google Analytics tag") is a JavaScript library for measuring how users interact with your website. The Google Analytics tag should be added near the top of the tag and before any other script or CSS tags, and add the property ID of the Google Analytics property you wish to work with. See https://developers.google.com/analytics/devguides/collection/analyticsjs/ {% endcomment %}
